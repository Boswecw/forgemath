# ForgeMath Canonical Equation Specification v1 — Initial

**Date:** April 2, 2026  
**Time Zone:** America/New_York

---

## Purpose

This document freezes the initial canonical equation family for ForgeMath.

It defines the recommended mathematical form for the first and second wave of governed lanes, using the already-established ecosystem variable vocabulary.

This is not yet the schema registry.
It is the lane-equation contract layer that later schema, DDL, API, and read-model work should implement.

---

## Governing Rules

1. Raw measurements remain visible and are not hidden behind synthetic scores only.
2. Bands and classifications are derived outputs, not replacements for raw values.
3. Every lane must use immutable parameter objects once persisted.
4. Every lane must be executable under deterministic runtime rules.
5. Every lane must emit explicit result status and replay posture.
6. Hybrid gate lanes must not be flattened into pure numeric lanes.
7. Improvement Yield must remain a portfolio-level doctrine surface, not false-precision incident vanity math.

---

## Normalization Helpers

### Bounded cap normalization

```text
cap_norm(x, cap) = min(x / cap, 1)
```

### Saturating recurrence transform

```text
sat(x, k) = 1 - exp(-k * x)
```

### Signed delta clamp

```text
delta_ratio(before, after, eps) = clamp((before - after) / max(before, eps), -1, 1)
```

### Rescale from [-1,1] to [0,1]

```text
z(x) = (x + 1) / 2
```

### Bounded union form

```text
bounded_union(a1 ... an) = 1 - Π(1 - ai)
```

# 1. Verification Burden Lane

## Purpose
Measure operational pain and cost generated by an incident, weak outcome, or validation burden.

## Admitted canonical inputs
- `implementation_minutes`
- `verification_minutes`
- `rework_minutes`
- `interruption_count`
- `downstream_fix_minutes`
- `uncertainty_band`

## Normalized factors

```text
I_n = cap_norm(implementation_minutes, I_cap)
V_n = cap_norm(verification_minutes, V_cap)
R_n = cap_norm(rework_minutes, R_cap)
X_n = cap_norm(interruption_count, X_cap)
D_n = cap_norm(downstream_fix_minutes, D_cap)
```

Map uncertainty band to scalar:

- `low = 0.10`
- `moderate = 0.35`
- `elevated = 0.65`
- `severe = 1.00`

## Canonical equation

```text
VB_raw = w_I*I_n + w_V*V_n + w_R*R_n + w_X*X_n + w_D*D_n + w_U*U_n
VB = clamp(VB_raw, 0, 1)
```

## Recommended default weights

```text
w_I = 0.15
w_V = 0.25
w_R = 0.25
w_X = 0.10
w_D = 0.10
w_U = 0.15
```

# 2. Recurrence Pressure Lane

## Purpose
Measure whether weakness is isolated, emerging, repeating, or structurally persistent.

## Admitted canonical inputs
- `recurrence_count_30d`
- `recurrence_count_90d`
- `same_system_recurrence_count`
- `cross_system_count`
- `post_control_recurrence_count`

## Normalized factors

```text
r30   = sat(recurrence_count_30d, k30)
r90   = sat(recurrence_count_90d, k90)
rsame = sat(same_system_recurrence_count, ksame)
rcross = sat(cross_system_count, kcross)
rpost = sat(post_control_recurrence_count, kpost)
```

## Canonical equation

```text
RP_raw = w30*r30 + w90*r90 + wsame*rsame + wcross*rcross + wpost*rpost
RP = clamp(RP_raw, 0, 1)
```

## Recommended default weights

```text
w30 = 0.25
w90 = 0.15
wsame = 0.20
wcross = 0.20
wpost = 0.20
```

# 3. Exposure Factor Lane

## Purpose
Measure how far the consequence reaches across public, operator, persistence, approval, and boundary surfaces.

## Admitted canonical inputs
- `public_exposure_flag`
- `operator_surface_flag`
- `persistence_truth_flag`
- `approval_surface_flag`
- `cross_system_flag`
- `local_cloud_boundary_flag`
- `severity_band`

## Core bounded-union equation

Let each flag be binary:

```text
f_pub, f_op, f_persist, f_approve, f_cross, f_boundary ∈ {0,1}
```

Then:

```text
EF_base = 1 - (1-α_pub*f_pub)
              * (1-α_op*f_op)
              * (1-α_persist*f_persist)
              * (1-α_approve*f_approve)
              * (1-α_cross*f_cross)
              * (1-α_boundary*f_boundary)
```

Severity uplift:

- `low = 0.00`
- `moderate = 0.08`
- `high = 0.16`
- `critical = 0.25`

## Canonical equation

```text
EF = clamp(EF_base + severity_uplift, 0, 1)
```

## Recommended default coefficients

```text
α_pub = 0.30
α_op = 0.15
α_persist = 0.25
α_approve = 0.15
α_cross = 0.20
α_boundary = 0.15
```

# 4. Reviewability Lane

## Purpose
Measure whether the system remains inspectable, traceable, evidence-grounded, and replay-safe enough for governed use.

## Lane class
**Hybrid gate lane**

This lane must not be implemented as scalar-only math.

## Hard gate conditions
The lane should emit `blocked` when any hard gate condition is true, such as:

- required evidence missing
- lineage broken
- compatibility unresolved
- replay minimum incomplete
- critical canonical artifact invalidated

## Optional posture score
Binary issue flags:

- `m_evidence`
- `m_lineage`
- `m_compat`
- `m_replay`
- `m_degraded`
- `m_invalid`

Then:

```text
RV_penalty = 1 - (1-β_evidence*m_evidence)
               * (1-β_lineage*m_lineage)
               * (1-β_compat*m_compat)
               * (1-β_replay*m_replay)
               * (1-β_degraded*m_degraded)
               * (1-β_invalid*m_invalid)

RV_score = 1 - RV_penalty
```

## Recommended default coefficients

```text
β_evidence = 0.30
β_lineage = 0.25
β_compat = 0.30
β_replay = 0.25
β_degraded = 0.15
β_invalid = 0.25
```

## Required emitted posture
- result status
- blocked / degraded / audit-only reason set
- optional numeric reviewability posture

# 5. Control Effectiveness Lane

## Purpose
Measure whether an approved hardening control actually reduced burden, recurrence, false-clean posture, and reviewability failure.

## Admitted canonical inputs
- `pre_control_burden_avg`
- `post_control_burden_avg`
- `pre_control_recurrence_rate`
- `post_control_recurrence_rate`
- `false_clean_before`
- `false_clean_after`
- `reviewability_failures_before`
- `reviewability_failures_after`

## Delta factors

```text
ΔB  = delta_ratio(pre_control_burden_avg, post_control_burden_avg, eps)
ΔR  = delta_ratio(pre_control_recurrence_rate, post_control_recurrence_rate, eps)
ΔF  = delta_ratio(false_clean_before, false_clean_after, eps)
ΔRV = delta_ratio(reviewability_failures_before, reviewability_failures_after, eps)
```

## Canonical equation

```text
CE_raw = wB*z(ΔB) + wR*z(ΔR) + wF*z(ΔF) + wRV*z(ΔRV)
CE = clamp(CE_raw, 0, 1)
```

## Recommended default weights

```text
wB = 0.25
wR = 0.35
wF = 0.20
wRV = 0.20
```

# 6. Governance Velocity Lane

## Purpose
Measure how quickly the ecosystem moves from recorded weakness to classification, proposal, approval, operationalization, and verified effect.

## Admitted canonical inputs
- `elapsed_record_to_classification`
- `elapsed_classification_to_control`
- `elapsed_control_to_approval`
- `elapsed_approval_to_operationalized`
- `elapsed_operationalized_to_verified`
- `stalled_state_flag`

## Stage timing transform
For each stage `i`:

```text
d_i = min(elapsed_i / T_i, d_max)
GV_i = exp(-λ_i * max(d_i - 1, 0))
```

## Canonical equation

```text
GV_base = Σ (w_i * GV_i)
GV = clamp(GV_base - stall_penalty, 0, 1)
```

Where:

```text
stall_penalty = γ_stall if stalled_state_flag = 1 else 0
```

# 7. Residual Governance Risk Lane

## Purpose
Measure how much danger remains after burden, recurrence, exposure, and control posture are all considered together.

## Admitted canonical inputs
- `verification_burden_raw` or normalized `VB`
- `recurrence_pressure_raw` or normalized `RP`
- `exposure_factor_raw` or normalized `EF`
- `control_effectiveness_raw` or normalized `CE`
- `control_gap_present`

## Canonical equation

```text
RGR_raw = θ1*RP + θ2*VB + θ3*EF + θ4*(1-CE)
RGR = clamp(RGR_raw + control_gap_uplift, 0, 1)
```

Where:

```text
control_gap_uplift = θ5 if control_gap_present = 1 else 0
```

## Recommended defaults

```text
θ1 = 0.30
θ2 = 0.20
θ3 = 0.25
θ4 = 0.20
θ5 = 0.10
```

# 8. Priority Score Lane

## Purpose
Produce an explainable prioritization surface for hardening review and portfolio sequencing.

## Admitted canonical inputs
- `RP`
- `VB`
- `EF`
- `RGR`
- `CE`
- `GV`
- `control_gap_present`
- measured improvement gain (`IG`)

## Canonical equation

```text
PS_raw = λ1*RP
       + λ2*VB
       + λ3*EF
       + λ4*RGR
       + λ5*(1-CE)
       + λ6*(1-GV)
       + λ7*G_gap
       - λ8*IG

PS = clamp(PS_raw, 0, 1)
```

Where:

```text
G_gap = 1 if control_gap_present = 1 else 0
```

## Recommended defaults

```text
λ1 = 0.18
λ2 = 0.14
λ3 = 0.14
λ4 = 0.22
λ5 = 0.10
λ6 = 0.10
λ7 = 0.07
λ8 = 0.05
```

# 9. Improvement Yield Lane

## Purpose
Measure whether governance and testing effort is producing meaningful ecosystem strengthening.

## Lane doctrine
Portfolio-level doctrine metric.
Not a fragile per-incident vanity score.

## Admitted canonical inputs
- `RP`
- `VB`
- `EF`
- `RGR`
- measured positive improvement deltas
- `governance_effort_proxy`

## Need and gain components

```text
IN = clamp(η1*RP + η2*VB + η3*EF + η4*RGR, 0, 1)
```

Use positive deltas only for gain:

```text
IG = clamp(μ1*max(ΔB,0) + μ2*max(ΔR,0) + μ3*max(ΔF,0) + μ4*max(ΔRV,0), 0, 1)
```

Normalize effort:

```text
GE = clamp(governance_effort_proxy / GE_cap, 0, 1)
```

## Canonical equation

```text
IY_raw = (IG / max(GE, eps)) * IN
IY = clamp(IY_raw, 0, 1)
```

